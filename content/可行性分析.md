# 一、研究设计（核心思路）

## 研究目的

基于关节活动度（ROM）及基础特征，构建舞者**分部位损伤风险预测模型**：

- 髋（Hip）
- 膝（Knee）
- 踝（Ankle）
- 腰（Lumbar）

👉 输出：4个部位分别的损伤概率

---

# 二、研究设计框架

## 1. 数据结构

每个受试者：

- 输入：ROM + 基础信息
- 输出：4个标签（是否发生损伤）

```
X → ROM + covariatesY → [Hip, Knee, Ankle, Lumbar] (0/1)
```
[[这里推荐三个方法]]
---
## 2. 特征变量

### ROM（核心）

- Hip：ER / IR / Flexion / Extension / Abduction
- Knee：Flexion / Extension
- Ankle：DF / PF
- Lumbar：Flexion / Extension / Rotation

### 控制变量

- 年龄
- 性别
- BMI
- 舞龄
- 训练时间
- 既往损伤

---

## 3. 标签定义

- 6–12个月内**time-loss injury**
- 按部位分别定义（Hip/Knee/Ankle/Lumbar）

# 三、建模方案

## 推荐主方案（最稳）

### ✔ 方案1：四个独立模型

- Logistic Regression（baseline）
- XGBoost（主模型）

👉 每个部位一个模型

---

## 可选高级方案

### ✔ 方案2：多标签模型

- Multi-output XGBoost / RF
- 或神经网络4输出

---

# 四、模型评估

每个部位分别评估：

- ROC-AUC（核心）
- Precision / Recall / F1
- PR-AUC
- Calibration（加分）

---

---

# 六、可行性总结

## ✔ 优点

- ROM易测量
- 结构清晰（4个部位）
- 易做临床解释
- 适合机器学习建模

---

## ⚠️ 主要问题

1. 样本量要求较高（≥200–300）
2. 轻伤定义容易偏差
3. 仅ROM可能信息不足
4. 类别不平衡（尤其腰/髋）
