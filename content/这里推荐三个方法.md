
## ✔ 方案1：四个独立二分类模型

对每个部位单独建模：

- Model_Hip
- Model_Knee
- Model_Ankle
- Model_Lumbar

### 优点

- 简单稳定
- 容易解释
- 审稿人容易接受
- 可分别分析风险因素

### 缺点

- 没有利用“部位间关联”

---

## ✔ 方案2：多标签学习（更高级，推荐发表用）

一个模型输出4个label：

```
Y = [Hip, Knee, Ankle, Lumbar]
```

算法可选：

- Multi-output Random Forest
- XGBoost multi-label wrapper
- Neural network (sigmoid 4 outputs)

### 优点

- 考虑损伤共现（非常重要）
- 更符合真实运动损伤机制

### 缺点

- 解释略复杂
- 数据要求更高
[[机器学习相对于纯统计方法有解释性问题]]
---

## ✔ 方案3：多任务学习（最高级，可选）

共享底层特征：

```
ROM + Covariates → shared layer → 4 task heads
```

适合：

- 样本较大（>500）
- 想投更高水平期刊
